Objective

Iterate in Shipyard development making it available as a standalone application and adding new functionalities.

Problem Statement

The visualization and exploration of big multidimensional datasets is not an easy labor, this has great complexity in design and data processing. Navio is a visualization widget that allows to do the mentioned tasks, however, currently there is no easy to use tool released for using Navio without the need of software development knowledge.

Deliverables

  • Desktop application with network and Navio coordinated visualization feature.
  • Standalone versions for Windows, Linux and MacOS
