Interactive and Interpretable Machine Learning
Towards more user-centric Machine Learning models and systems

Master project by Fabian Camilo Peña working under the supervision of John Alexis Guerra Gómez





Objective

Design and implement an interactive Machine Learning framework including interface elements such as sample review, feedback assignment and model inspection (interpretation) with the purpose of improving high-level user tasks in an iterative way. Evaluate the framework by case studies based on tabular data.

Problem Statement

Black-box Machine Learning (ML) is a term commonly applied to the use of ML models that aren't completely understood by users entailing lack of trust and, therefore, hindering decision making. This disadvantage increases with the use of newfangled technologies such as Deep Learning where more parameters need to be adjusted requiring large amounts of data. In black-box ML models, users do not have anything to say about the produced results and the system does not take into account the user’s expert feedback that could greatly improve the overall performance of the system. As Lipton says: “While the ML objective might be to reduce error, the real-world purpose is to provide useful information”.
